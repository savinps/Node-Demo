VIOS INSTALLATION FORM


Please read the guidelines before submitting the form.

Guidelines

  1. Make sure that HMC and VIOS are UP before submitting the form
  2. Create or Update the VIOS details with correct HMC and Managed System name before submitting the form
  3. Check the result of Ping Test after submitting the form.
  4. For VIOS which are down default installation will be done on hdisk0. Click the proper option once you get the message.
  5. While filling the rootvg disk field, please enter the value in the format viosname_rootvgDisk (eg. beev1_hdisk).
  6. After confirming the installation procedure, go to VIOS Install Table to see the installation logs.
